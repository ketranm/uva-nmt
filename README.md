# UvA-NMT
University of Amsterdam's Neural Machine Translation

## Features

* Seq2seq with attention
* Fast LSTM / enable `cudnn`
* Hard Attention
* REINFORCE
* [[Conditional Attention GRU](https://github.com/nyu-dl/dl4mt-tutorial/blob/master/docs/cgru.pdf)]
